and nervous system perception.

perceptive activation conditions.

materialised field difference.

part of the field ontology.

only through differential activation.

no behavioural feedback.

no new relational modification.

future relational events.

primitive structure

how recursive topology becomes perceptible
without becoming semantic psychology.
